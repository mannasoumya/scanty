# Hello! scanty is here
## A basic Console Client written in Python for Windows
### Double-Click on the Executable(scanty.exe) to Run

Source is a single operable file **scanty.py** 

**Currently supported commands:**
```console
cd
cls
commands
exit
help
ls
lsdir
pwd
setprompt
```
